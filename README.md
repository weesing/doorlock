# Required Arduino libraries
- `Adafruit GFX`
- `Adafruit SSD1306`
## How to Install
- Go to Arduino IDE
- Under `Tools` -> `Library Manager`
- Search for `Adafruit GFX` and `Adafruit SSD1306`
- Click on `Install`
